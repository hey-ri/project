# project

- vue 폴더 간단 설명
  > componeet 폴더는 재사용이 가능한 ui를 담아두는 폴더입니다.
  > views 폴더는 하나의 페이지를 만들어 담아두는 폴더입니다.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
